# 레전드 오브 대철 (대철의 전설)

대철중학교를 홍보하기 위해 만들어진 프로젝트입니다.

저희 학교는 작지만, 아름다운 학교입니다.

하지만 그걸 알리기 위해서 만들었다고 하면 거짓말이겠죠.

사실 학교 부수고 싶어서 만들었습니다.

# 게임요소

* 거의 모든 것을 파괴할 수 있습니다.
* 멋진 망치 (블렌더로 제작)
* 실제 학교랑 거의 똑같습니다. (완전히 똑같지는 않음)
